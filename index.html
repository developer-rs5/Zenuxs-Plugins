Installation Guide

Follow these steps to install any plugin:

  1. Download the plugin `.jar` file for your desired plugin.
  2. Place the `.jar` file into your server's `/plugins` directory.
  3. Restart or reload the server.
  4. Modify the plugin settings through the config file if needed.
